# Guía de Ejecución - Zuliadog Web

Para ejecutar este proyecto localmente y abrirlo correctamente en Google Chrome, sigue estos pasos:

### 1. Iniciar el servidor de desarrollo
Desde la terminal en la raíz del proyecto, ejecuta:

```bash
npm run dev
```

### 2. Abrir en el navegador
Una vez que veas el mensaje `Ready`, abre Google Chrome y navega a la siguiente dirección:

**URL:** [http://localhost:9002](http://localhost:9002)

---

**Nota:** Este proyecto está configurado permanentemente para el puerto **9002**. Si intentas entrar por el puerto estándar 3000, no funcionará.
